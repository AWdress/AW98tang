## 📱 移动端适配

在手机上访问时，界面会自动调整：
- 侧边栏隐藏，通过顶部☰按钮打开
- 卡片布局自动调整为单列
- 按钮和文字自动放大，便于触摸

## 📊 四大功能模块

### 1. 仪表盘（Dashboard）
**功能**：实时监控系统状态

**显示内容**：
- 📊 运行状态（运行中/已停止）
- 💬 今日回复数量和限额
- ✅ 今日签到状态和时间
- ⚙️ 当前配置信息

**操作按钮**：
- ▶️ 启动机器人
- ⏹️ 停止机器人
- ⚙️ 系统配置（跳转到配置页面）

### 2. 回复记录（Replies）
**功能**：查看所有历史回复

**显示格式**：
```
┌──────────────────────────────────────┐
│ 🕐 2025-10-04 15:30:00              │
│                                      │
│ 📌 DFE-106 性感暴发工E反りアクメ... │
│     (点击标题可跳转到帖子)           │
│                                      │
│ 💬 激狭美穴让人心痒难耐，真想亲身   │
│    体验那种紧致的快感！              │
└──────────────────────────────────────┘
```

**特点**：
- 显示最近1000条回复记录
- 按时间倒序排列（最新的在最上面）
- 卡片式设计，悬停有高亮效果
- 点击标题直接跳转到论坛帖子

### 3. 运行日志（Logs）
**功能**：实时查看系统运行日志

**特点**：
- 最多显示500条日志
- 彩色显示不同级别：
  - 🔵 INFO - 蓝色（普通信息）
  - 🟡 WARNING - 黄色（警告）
  - 🔴 ERROR - 红色（错误）
  - 🟢 SUCCESS - 绿色（成功）
- 自动滚动到底部（显示最新日志）
- 手动滚动后停止自动滚动
- 提供"⬆️ 顶部"和"⬇️ 底部"快速定位按钮

### 4. 系统配置（Config）
**功能**：在线修改所有配置参数

**可配置项**：
- 每日回复上限（1-100）
- 回复间隔-最小值（≥30秒）
- 回复间隔-最大值（≥60秒）
- 启用自动回复（是/否）
- 启用每日签到（是/否）
- 启用定时任务（是/否）
- 定时运行时间（如：03:00）
- 测试模式（是/否）

**操作按钮**：
- 💾 保存配置
- 🔄 重置（重新加载当前配置）

## 🚀 快速上手

### 桌面端使用
1. 访问 `http://localhost:5000`
2. 登录（默认账号：admin / password）
3. 在左侧菜单点击不同模块切换页面
4. 在仪表盘点击"启动机器人"开始运行

### 移动端使用
1. 在手机浏览器访问 `http://你的IP:5000`
2. 登录
3. 点击左上角☰按钮打开菜单
4. 选择要查看的模块
5. 点击屏幕其他区域关闭菜单

## 💡 使用技巧

### 查看回复详情
1. 点击左侧菜单"💬 回复记录"
2. 滚动浏览所有历史回复
3. 点击帖子标题直接跳转到论坛

### 监控运行状态
1. 留在"📊 仪表盘"页面
2. 观察运行状态指示灯（绿色=运行中，红色=已停止）
3. 查看今日回复数和签到状态

### 查看实时日志
1. 切换到"📋 运行日志"页面
2. 日志会自动滚动显示最新内容
3. 手动滚动查看历史日志
4. 点击"⬇️ 底部"回到最新日志

### 修改配置
1. 切换到"⚙️ 系统配置"页面
2. 修改需要调整的参数
3. 点击"💾 保存配置"
4. 等待提示"配置保存成功"

## 🎯 数据实时更新

### 更新频率
- 运行状态：每3秒更新
- 日志信息：每2秒更新
- 回复记录：每5秒更新

### 自动刷新
所有数据都会自动刷新，无需手动刷新页面。

## 🔒 安全说明

### 修改登录密码
创建 `.env` 文件：
```env
WEB_USERNAME=你的用户名
WEB_PASSWORD=你的密码
```

### 远程访问
如需在其他设备访问：
1. 确保防火墙开放5000端口
2. 使用局域网IP访问：`http://192.168.x.x:5000`
3. **不要在公网暴露控制面板**

## 📊 回复记录说明

### 数据存储
- 所有回复保存在 `data/stats.json` 中
- 最多保留1000条历史回复
- 每日数据会归档到历史记录

### 显示规则
- 按时间倒序排列
- 显示完整帖子标题
- 显示实际回复内容
- 提供帖子链接（可点击）

## ⚠️ 注意事项

1. **不要在机器人运行时修改配置**
   - 先停止机器人
   - 修改配置
   - 重新启动

2. **移动端操作**
   - 建议使用Chrome或Safari浏览器
   - 横屏可以获得更好的体验

3. **回复记录加载**
   - 首次使用可能没有数据
   - 运行一次机器人后会显示记录

4. **日志滚动**
   - 手动滚动后会停止自动滚动
   - 点击"⬇️ 底部"恢复自动滚动
